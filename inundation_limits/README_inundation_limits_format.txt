PTHA inundation-limit database

Each grid has:
  - one flat JSON file with inundation-limit contours
  - one diagnostic PNG with all return-period contours over the DEM

Return periods: 72, 100, 200, 475, 975, 2475, 3000 years
Inundation threshold: FlowDepth >= 0.03 m
Dry-land definition: DEM > 0 m rel. MHW
Contour meaning: landward inundation limit only; shoreline contour excluded
